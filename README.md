## Requisitos

Para poder ejecutar el videojuego, es necesario tener instalado lo siguiente:

- `g++`

- `c++ 17`

- Raylib

## Instrucciones de Uso

En la carpeta del proyecto, abra una terminal e ingrese `make clean; make` y `make run`.
